# VEHICLE MANAGER (by vu)

a program that help you manage your vehicles.

## FEATURES

* Every vehicle is stored with their name and their wheels count
* A default "Car - 4" is added as the first vehicle
* Can add vehicles
* Can edit vehicles
* Can delete, delete all
* Can insert between vehicles
* And more to come!

## HOW TO USE

* Add vehicles by typing in the input box, in the format "{name} {wheels}"
* Example: Bus 6 -> Bus - 6
* Edit, insert and so on, you must select the vehicle you want to edit, then input edit. insert would add the inputted vehicle AFTER the selected vehicle
